## ‚ö° What is RAWviewer Lite?
**RAWviewer Lite** is a streamlined, ultra-fast image viewer designed for rapid navigation of standard image formats. It strips away heavy features like RAW processing, gallery view, and deep zooming to focus purely on speed and simplicity.

This version is ideal for users who need to quickly browse through folders of JPEGs or PNGs without any overhead.

## ‚ú® Lite Features

- **Cross-platform support**: Windows and macOS
- **Instant Performance**: Minimized footprint for extremely fast load times.
- **Random Navigation**: Support for randomized image traversal.
- **Standard Format Support**: Fast viewing of JPG, PNG, WEBP, and TIFF.
- **Simplified Controls**: No complex modes - just view and navigate.
- **Portable**: No installation required.
